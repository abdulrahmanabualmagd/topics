1. <link>       : Defines the relationship between a document and an external resource, commonly used to link stylesheets (CSS) to HTML documents.
2. <script>     : Embeds or links to a client-side script, typically JavaScript, allowing for dynamic interaction and behavior within the web page.
3. <style>      : Defines embedded CSS styles within an HTML document, allowing for inline styling of elements without the need for an external stylesheet.
